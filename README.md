# 口袋48成员直播视频流抓取

口袋48下载成员的直播视频。  
抓取视频使用了ffmpeg，开发者请自行下载并放到文件夹内。

---

## 文件打包方法
* 将文件夹内的文件打包成压缩文件，并重命名为*.nw
* 将*.nw复制到和nw.exe同级目录下，运行
```
copy /b nw.exe+*.nw *.exe
```
* 双击*.exe即可运行程序
